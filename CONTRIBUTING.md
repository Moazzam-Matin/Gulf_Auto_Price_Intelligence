# Contributing to Gulf Auto Price Intelligence

First off, thank you for considering contributing!

### How Can I Contribute?

1. **Feature Requests:** Suggest new data sources (e.g., Dubizzle or YallaMotor scraping logic).
2. **Bug Reports:** If the `engine.py` crashes on a specific car model, open an Issue.
3. **Data Enrichment:** We are specifically looking for a way to map 'GCC' vs 'Import' specs.
4. **Code Optimization:** Help us improve the Random Forest hyperparameters.

### Pull Request Process

1. Fork the repo and create your branch (`git checkout -b feature/AmazingFeature`).
2. Ensure your code follows the existing structure in `research.ipynb`.
3. Update the `requirements.txt` if you add new libraries.
4. Submit a Pull Request with a detailed description of your changes.
